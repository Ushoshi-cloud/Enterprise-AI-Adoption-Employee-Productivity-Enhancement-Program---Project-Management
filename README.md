Developed a 4-page Power BI PMO dashboard solution to monitor 72 project tasks, 15 risks, 10 issues, $960K budget allocation, and 18 project resources for an Enterprise AI Adoption Program, enabling executive-level tracking of project health, schedule performance, risk exposure, and resource utilization.
# Enterprise AI Adoption & Employee Productivity Enhancement Program

## Project Overview
This project simulates the planning, execution, monitoring, and governance of an Enterprise AI Adoption & Employee Productivity Enhancement Program. The objective was to demonstrate Project Management Office (PMO) capabilities by creating project management artifacts and transforming them into interactive Power BI dashboards for executive reporting and decision-making.

The program focuses on implementing AI-powered solutions across business functions to improve productivity, automate manual processes, enhance decision-making, and strengthen project governance.

## Business Objective
The primary objectives of this program are:
- Improve employee productivity through AI-enabled workflows
- Reduce manual effort across operational processes
- Enhance organizational decision-making using AI tools
- Strengthen project governance and reporting
- Monitor budget, resources, risks, issues, and milestones through data-driven dashboards

## Dashboard Suite

### Executive Project Health Dashboard

Provides a high-level view of overall project health and performance.

#### Key Metrics
- Total Tasks
- Completed Tasks
- Open Risks
- Open Issues
- Average Resource Utilization

#### Visualizations
- Task Status Distribution
- Tasks by Phase
- Interactive Filters (Phase, Owner, Status)

### Schedule & Milestone Performance Dashboard

Tracks project execution progress and task completion across phases.

#### Key Metrics
- Total Tasks
- Completed Tasks
- In Progress Tasks
- Project Completion Percentage

#### Visualizations
- Tasks by Phase
- Tasks by Owner
- Task Status Distribution by Phase
- Interactive Filters (Phase, Owner)

### Risk & Issue Management Dashboard

Monitors project risks and issue resolution effectiveness.

#### Key Metrics
- Total Risks

- Total Issues
- Open Issues

#### Visualizations

- Risks by Status
